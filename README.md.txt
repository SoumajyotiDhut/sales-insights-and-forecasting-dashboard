# 🧠 Sales Forecasting Dashboard (Power BI Project)

## 📌 Objective
To contribute to business success by utilizing data analysis and time series forecasting techniques through Power BI dashboards. The project aims to uncover sales trends, provide strategic insights, and accurately predict sales for the next 15 days.

---

## 🗂️ Project Description

The objective is broken down into the following key components:

1. **Dashboard Creation**  
   - Designed an interactive and visually appealing Power BI dashboard.
   - Incorporated KPIs like Total Sales, Total Orders, and Total Profit.
   - Enabled filtering, drill-throughs, and slicers for multi-level data analysis.

2. **Data Analysis**  
   - Performed detailed analysis on sales performance across regions, categories, and customer segments.
   - Used charts and aggregations to highlight high-performing areas and identify bottlenecks.

3. **Sales Forecasting**  
   - Used time series analysis on historical sales data.
   - Forecasted sales for the next 15 days using Power BI's built-in forecasting tools.
   - Optional: Integrated Python (ARIMA/Prophet) for advanced forecasting.

4. **Actionable Insights and Recommendations**  
   - Generated insights to improve sales strategy.
   - Highlighted trends that can assist in strategic decision-making for better growth and customer satisfaction.

---

## 🧰 Tools & Technologies Used

- 📊 Power BI (Visualization, DAX)
- 📁 CSV Dataset (SuperStore Sales)
- 📈 Time Series Forecasting (Power BI Forecast)
- 🧮 Microsoft Excel (Data cleaning)
